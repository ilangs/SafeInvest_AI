> 금융감독원 e-금융교육센터 공식 콘텐츠 기반 AI 금융 교육 플랫폼

**잃지 않는 투자의 시작, 당신만의 AI 커스텀 대시보드**

---

## 📌 프로젝트 개요

SafeInvest AI는 한국 금융감독원의 공식 교육 콘텐츠를 활용하여 사용자에게 인생 단계별 맞춤 학습 경로를 제공하는 AI 금융 교육 플랫폼입니다.

### 핵심 가치

- **✨ 기본기 자체 제작 영상**: SafeInvest AI가 직접 만든 10편 영상으로 빠른 입문
  - 입문 패키지 (5편): PER, 손익계산서, 강세장/약세장, 분산투자, 실적 발표
  - 통합 가이드 (5편): 주식 투자 5편으로 시작하기 (왜→무엇→언제→어떻게→누가)
- **🎯 인생 단계별 학습 경로**: 청소년부터 은퇴 후까지 12개 코스로 체계화
- **📚 금감원 공식 콘텐츠**: 1,345건의 검증된 영상·도서·웹툰·오디오북 (24개 주제)
- **🤖 콘텐츠 + AI 튜터 동시 사용**: 학습 중 바로 질문하고 답변받는 학습 도우미

## 🏗 시스템 아키텍처

### 데이터 흐름

```
1,345건 (카테고리 다중 매칭)
    ↓
802 고유 콘텐츠
    ↓
24개 교육 주제 × 14개 교육대상
    ↓
12개 학습 경로의 주차별 매칭 (3단계 엄격 모드)
    ↓
인생 단계별 추천 + AI 튜터 학습
```

### 매칭 알고리즘 (3단계)

1. **Step 1 (엄격)**: 풀(주제) + 교육대상 + 학습목표 키워드 모두 매칭
2. **Step 2 (중간)**: 풀 + 교육대상 (키워드 풀음)
3. **Step 3 (느슨)**: 풀만 (교육대상도 풀음)

부족할 때만 다음 단계로 완화. 174 슬롯 모두 100% 채움 (placeholder 0).

### 기술 스택

```
Frontend: Vanilla JS + 모듈형 위젯 (단일 HTML 파일)
Backend:  FastAPI (uvicorn)
LLM:      OpenAI API (LangChain RAG 준비)
Vector:   Supabase (pgvector) - 향후 통합
Data:     금감원 e-금융교육센터 API
```

## 🚀 실행 방법

### 사전 준비

```bash
# 1. Python 가상환경 생성
python -m venv venv
venv\Scripts\activate   # Windows
# source venv/bin/activate  # Mac/Linux

# 2. 의존성 설치
pip install -r requirements.txt

# 3. 환경 변수 설정
copy .env.example .env
# .env 파일 열어서 FSS_API_KEY, OPENAI_API_KEY 입력
```

### 데이터 준비

`data/real_contents.json` (28MB, 1,345건)이 필요합니다. 
파일이 없다면 다음 명령으로 수집:

```bash
python scripts/collect_real_data.py
```

### 서버 실행

```bash
python -m uvicorn app.main:app --reload
```

브라우저에서 `http://127.0.0.1:8000` 접속.

## 🎨 주요 기능

### 1. ✨ 기본기 - 자체 제작 입문 영상 (v2.10 NEW)

- **두 시리즈, 총 10편**:
  - **입문 패키지 (5편)**: 각 카테고리 1편씩, 핵심 개념 깊이 다룸
  - **통합 가이드 (5편)**: 주식 투자 5편으로 시작하기 (왜→무엇→언제→어떻게→누가)
- **6개 카테고리**: 재무 지표 / 재무제표 / 시장 기초 / 투자 원칙 / 뉴스 해석 / 통합 가이드
- **HTML5 비디오 플레이어**: 자체 서버 호스팅 (`/static/videos/`)
- **재생 속도 조절**: 0.75x / 1x / 1.25x / 1.5x / 2x 5단계
- **AI 사이드 챗봇**: 영상 시청 중 즉시 질문 가능 (학습 목표 기반 답변)
- **저작권 자체 보유**: 자유로운 활용·배포 가능 (외부 콘텐츠와 분리)

### 2. 학습 경로 (메인 동선)

- **3문항 설문** → 인생 단계 + 주요 고민 + 시간 → 12개 중 추천
- **12개 카드 그리드**: 각 카드에 주차별 theme 미리보기, 직접 선택 가능
- **학습 흐름 기반 배치**: 인생 시작 → 투자 학습 → 자산·노후 → 위기 대응

### 3. 교육 센터 (자유 탐색)

- **6개 금감원 공식 분류**로 24개 주제 그룹핑
- **3종 필터**: 콘텐츠 유형 / 교육 대상 / 시간
- 콘텐츠 카드 → 상세 페이지 진입

### 4. AI 튜터 (콘텐츠 + 챗봇)

- **콘텐츠 시청 중 우측 사이드 챗봇**
- 콘텐츠 메타데이터 기반 contextual 답변 (`/api/chat/contextual`)
- 추천 질문 자동 생성 (`/suggested-questions`)

### 5. 재생불가 콘텐츠 AI 우회

금감원 API의 한계로 일부 콘텐츠는 직접 재생/열람 불가:
- 영상 (makeTypeCode=1): 일부는 fileDown 메타데이터만 제공, iframe 거부
- 웹툰 (makeTypeCode=3): 거의 전부 금감원 호스팅 + iframe 거부

**처리 방향**:
- 백엔드: `is_playable` + `learning_mode: "watch" | "ai_summary"` 자동 판정
- UI: 카드에 🤖 AI 학습 배지 + 보라색 CTA
- 상세 페이지: 재생 불가 콘텐츠 → AI 학습 카드 + 추천 질문 자동 로드
- YouTube 임베드 거부 시: "🤖 재생 안 되시나요? AI로 학습" 폴백 버튼

→ "콘텐츠 + AI 튜터 동시 사용" 킬러 기능과 정합

## 📜 라이선스 및 저작권

### 콘텐츠 저작권

본 플랫폼이 제공하는 학습 콘텐츠의 저작권은 **각 제작기관**에 귀속됩니다.

금융감독원 e-금융교육센터는 다양한 금융교육기관의 콘텐츠를 모아 제공하는 **포털**이며,
본 플랫폼은 그 OpenAPI를 통해 콘텐츠 정보를 제공받습니다.

데이터에 포함된 주요 제작기관: 금융위원회, 금융감독원, 한국거래소(KRX), 한국예탁결제원,
한국주택금융공사, 신용회복위원회, KBS 등 총 15개 이상 기관

### 본 플랫폼의 활용 범위

✅ **활용하는 것**:
- 콘텐츠 메타데이터 (제목, 요약, 분류, 대상)
- 외부 콘텐츠 URL (YouTube 영상, 기관 사이트 링크)
- 파일 다운로드 링크 (금감원 fileDown API)
- AI 튜터의 메타데이터 기반 답변

❌ **하지 않는 것**:
- 콘텐츠 본문 추출/인용/재구성 (2차 편집 금지)
- PDF 텍스트 추출 후 가공
- 영상 자막 추출 후 AI 학습
- 콘텐츠 우리 서버에 호스팅
- 상업적 이용 (수익 창출, 광고)

### 운영 형태

본 플랫폼은 **비영리 학생 졸업 프로젝트**로 운영됩니다.

- 사용자에게 비용을 받지 않음
- 광고 게재 없음
- 학교 평가 및 포트폴리오 용도

### 향후 상업화 가능성

현재 데이터 구성에서 상업적 이용을 위해서는 다음 절차가 필요합니다:
- 각 콘텐츠 제작기관과 사전 협의 (15개 이상 기관)
- 또는 자체 콘텐츠로 대체

이는 본 학생 프로젝트의 범위를 벗어나며, 코드 공유 시 동일한 책임 경계가 적용됩니다.
